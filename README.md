# HR Analytics Dashboard | Power BI

## Objective
Analyze employee attrition patterns to help HR teams 
identify at-risk segments and reduce turnover.

## Dataset
IBM HR Analytics Attrition Dataset (Kaggle)
- 1,470 rows | 35 columns

## Tools Used
- Power BI Desktop
- DAX (calculated measures)
- Power Query (data transformation)

## Dashboard Pages
1. Executive Summary
2. Employee Demographics
3. Job and Performance
4. Satisfaction Analysis

## Key Insights
- Overall attrition rate: 16.12%
- Laboratory Technicians have highest attrition
- Single employees leave more than married
- Employees with overtime have higher attrition
- Low salary employees leave more
- Life Sciences has highest attrition by education field
- Employees in age group 26-35 leave the most

## Skills Demonstrated
- DAX measures (COUNTROWS, FILTER, DIVIDE, AVERAGE)
- Power Query (conditional columns)
- Multi-page dashboard design
- Slicers for interactive filtering
- Multiple chart types
